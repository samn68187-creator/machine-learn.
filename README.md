# Marketing Funnel Analysis Dashboard

## Project Overview
This project analyzes the marketing funnel to measure campaign performance and customer conversions using Power BI.

## Tools Used
- Power BI
- Microsoft Excel / CSV
- DAX
- Power Query

## Key KPIs
- Total Impressions
- Total Clicks
- Click Through Rate (CTR)
- Total Leads
- Lead Conversion Rate
- Total Customers
- Customer Conversion Rate

## Files
- marketing funnel analysis dashboard.pbix
- marketing_funnel_data.csv

## Dashboard
Open the .pbix file using Power BI Desktop.
